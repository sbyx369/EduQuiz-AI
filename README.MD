# EduQuiz AI

## Overview
EduQuiz AI is an AI-powered syllabus quiz generator built using IBM watsonx.ai, IBM Granite Models, and LangFlow.

The system generates:
- Learning Modules
- Multiple Choice Questions (MCQs)
- Correct Answers
- Explanations

from any educational topic automatically.

## Tech Stack
- IBM watsonx.ai
- IBM Granite Foundation Models
- LangFlow
- Python
- Prompt Engineering

## Features
- Automated Quiz Generation
- Learning Module Creation
- Answer Generation
- Explanation Generation
- Multi-subject Support

## Architecture
User Topic → LangFlow → Prompt Template → IBM Granite Model → Quiz Generation → Output
