# EduQuiz AI

## Overview

EduQuiz AI is an AI-powered educational assistant built using IBM watsonx.ai, IBM Granite Models, and LangFlow.

The system automatically generates:

* Learning Modules
* Multiple Choice Questions (MCQs)
* Correct Answers
* Explanations

from any educational topic entered by the user.

## Technologies Used

* IBM watsonx.ai
* IBM Granite 4
* LangFlow
* Prompt Engineering
* Generative AI

## Workflow

User Topic → LangFlow → IBM Granite → Quiz Generation

## Features

* Automated Quiz Generation
* Learning Module Creation
* Answer Explanations
* Multi-Subject Support
